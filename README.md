# BaselineDocs Marketing Site

Complete Next.js marketing site for www.baselinedocs.com

## Quick Start

```bash
pnpm install
cp .env.example .env.local
# Edit .env.local with your values
pnpm dev
```

Open http://localhost:3001

## Features

- Landing page with hero + features
- Pricing page with 3 tiers  
- Full signup flow with validation
- Stripe customer creation
- Tenant provisioning
- Real-time subdomain checking

## Deploy

```bash
vercel --prod
```

See DEPLOYMENT-READY.md for full instructions.
